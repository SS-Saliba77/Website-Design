Detailed Description

Game History

Wuthering Waves is a 2024 free-to-play open world action role-playing game developed and published by Kuro Games. First revealed on March 23, 2022, it was initially inspired by Death Stranding with the aim to create a post-apocalyptic world where players can experience the fusion of new and old civilizations.
